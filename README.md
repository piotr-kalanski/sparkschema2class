# Getting started

## Include dependencies

```scala
"com.github.piotr-kalanski" % "sparkschema2class_2.11" % "0.1.0"
```

or

```xml
<dependency>
    <groupId>com.github.piotr-kalanski</groupId>
    <artifactId>sparkschema2class_2.11</artifactId>
    <version>0.1.0</version>
</dependency>
```
